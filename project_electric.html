Подробнее о проекте "Моделирование работы электрической цепи"

Стек проекта

  • Java, Java Core, Maven

Для реализации программы моделирования работы электрической цепи были использованы следующие технологии:

Backend

  • Commons Math: The Apache Commons Mathematics Library - это библиотека легких, автономных математических и статистических компонентов, предназначенных для решения наиболее распространенных задач, недоступных в языке программирования Java или Commons Lang.
  • Расширенный узловой метод для механических систем - метод, позволяющий расчитать значения параметров системы в текущий момент времени
  • Maven - фреймворк для автоматизации сборки проектов на основе описания их структуры в файлах POM (англ. ), на языке XML.

Процесс разработки

Разработка и развертывание сайта для туристического агентства включили в себя несколько ключевых этапов:

  • Изучение. На этом этапе изучался расширенный узловой метод для механических систем.
  • Проектирование. На этом этапе строилась диаграмма классов и проектировался дизайн системы.
  • Разработка. Написание кода для реализации метода моделирования работы электрической системы. Был разработан общий класс для реализации схемы, а также класс решателя, от которого наследовались другие классы с реализуемыми методами.
  • Тестирование. Тестирование проводилось путем сравнения результатов работы программы с результатами работы специализированного и надежного ПО.
